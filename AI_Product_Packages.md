## 1. 双轨制交付策略概述

我们采用"双轨制"策略，针对不同客户群体提供不同的交付物：

```
┌─────────────────────────────────────────────────────────────────┐
│                        客户类型分流                              │
└─────────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
┌─────────────────────────┐     ┌─────────────────────────┐
│   普通客户 / 低预算      │     │   高端客户 / 复杂需求    │
│   ($100-$500)           │     │   ($500-$2000+)         │
│   非技术人员为主         │     │   有技术团队或愿意付费   │
└─────────────────────────┘     └─────────────────────────┘
              │                               │
              ▼                               ▼
┌─────────────────────────┐     ┌─────────────────────────┐
│   交付物: FastGPT 应用   │     │   交付物: Huice Agent   │
│   ✅ 可视化 Flow 配置    │     │   ✅ LangGraph 多步推理  │
│   ✅ 客户可自己维护      │     │   ✅ MCP 工具集成        │
│   ✅ 嵌入式部署          │     │   ✅ 知识图谱 RAG        │
│   ✅ 品牌认知度高        │     │   ✅ 定制化开发          │
└─────────────────────────┘     └─────────────────────────┘
```

## 2. 轨道 A: FastGPT 产品方案

### 2.1 产品一：AI 客服机器人 (Basic)

**价格**: $150-$300
**交付周期**: 3-5 天
**目标客户**: 小型企业、创业公司

**交付内容**:
- 1 个 FastGPT 应用
- 知识库配置（最多 50 个 FAQ）
- 嵌入代码（可放入官网）
- 基础培训文档

**客户可自己做**:
- 添加/修改 FAQ
- 调整回复风格
- 查看对话统计

### 2.2 产品二：企业知识库 Bot (Standard)

**价格**: $300-$500
**交付周期**: 5-7 天
**目标客户**: 中型企业、SaaS 公司

**交付内容**:
- 1 个 FastGPT 应用
- 知识库配置（最多 200 个文档）
- 可视化 Flow 配置
- 嵌入代码 + API 接入指南
- 1 小时培训视频

**客户可自己做**:
- 上传新文档
- 调整 Flow 逻辑
- 管理多个知识库
- 查看详细统计

### 2.3 产品三：多场景 AI 助手 (Premium)

**价格**: $500-$800
**交付周期**: 7-10 天
**目标客户**: 大型企业、需要多个 Bot

**交付内容**:
- 3-5 个 FastGPT 应用（不同场景）
- 完整知识库体系
- 高级 Flow 配置（含条件分支）
- n8n 工作流集成
- 2 小时培训 + 30 天支持

**客户可自己做**:
- 管理所有应用
- 调整任意 Flow
- 配置自动化规则

## 4. 混合模式产品

### 4.1 产品七：FastGPT + Huice 集成方案

**价格**: $800-$1500
**交付周期**: 14-21 天
**目标客户**: 需要简单界面 + 高级能力

**交付内容**:
- FastGPT 应用（前端）
- Huice Agent（后端）
- 集成配置
- 完整文档

**工作原理**:
```
用户 → FastGPT (简单问答)
         │
         │ 遇到复杂问题
         ▼
      FastGPT Flow (HTTP 节点)
         │
         │ 调用 Huice API
         ▼
      Huice Agent (复杂推理)
         │
         │ 返回结果
         ▼
      FastGPT (展示给用户)
```

## 6. 基础设施概况

### 6.1 轨道 A: FastGPT Stack

- **FastGPT** (demo.yansemei.com) - 知识库与对话应用
- **OneAPI** (api.yansemei.com) - 模型路由与配额管理

### 6.2 轨道 B: Huice Core

- **Agent Service** (agent.yansemei.com) - LangGraph Agent
- **RAG Core** (kb.yansemei.com) - LightRAG 知识图谱
- **Chat UI** (aurora.yansemei.com) - Agent 对话界面
- **Admin UI** (chat.yansemei.com) - 知识库管理

### 6.3 共享基础设施

- **n8n** (flow.yansemei.com) - 工作流自动化
- **NPM** - 反向代理 + SSL
- **SiliconFlow** - 云模型 (DeepSeek-V3 + Qwen Embedding)
